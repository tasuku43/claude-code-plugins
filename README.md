# claude-code-plugins

Personal [Claude Code](https://claude.ai/code) marketplace repository by [@tasuku43](https://github.com/tasuku43).

## Plugins

| Plugin | Description |
|--------|-------------|
| [cc-maintenance](./plugins/cc-maintenance/) | Maintain and optimize Claude Code settings, context efficiency, and skill/command definitions |

## Installation

Add this marketplace in Claude Code:

```
plugin marketplace add tasuku43/claude-code-plugins
```

Then install a plugin:

```
plugin install cc-maintenance
```
